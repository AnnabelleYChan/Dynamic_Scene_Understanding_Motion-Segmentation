# Dynamic_Scene_Understanding_Motion-Segmentation
A computer vision pipeline for analyzing motion in video by combining optical flow, geometric modeling, and segmentation. The system separates camera motion from independently moving objects using Focus of Expansion (FOE) estimation via RANSAC, dense optical flow (Farnebäck), and SAM3-based object segmentation with IoU-based tracking across frames.

The project enables robust scene understanding under complex motion, occlusions, and scale changes by integrating classical vision techniques with modern foundation models.
